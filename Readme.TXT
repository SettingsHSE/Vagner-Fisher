В данной задаче вы должны реализовать поиск расстояния Левенштейна между строками с помощью более эффективного алгоритма Вагнера-Фишера.

Формат ввода
В первой строке число N - количество пар строк, которые надо сравнить и вычислить расстояния Левенштейна. 
Затем идет N*2 строк, которые и надо сравнить попарно. 

Формат вывода
Необходимо вывести минимальное расстояние.
Весь ответ в одной строке. N чисел разделенных пробелами, каждое число это расстояние Левенштейна для пары строк. 
Расстояния должны соответствовать порядку пар строк для сравнения (1-е число - расстояние для 1-й пары, 2-е число - расстояние для 2-й пары...).
